### Problem 0.6 — Palindrome

**Input file:** input.txt  
**Output file:** output.txt  
**Time limit:** 1 s  
**Memory limit:** no

You are given a non-empty string `S` consisting of lowercase Latin letters.  
The length of `S` does not exceed 7000 characters.

Your task is to remove the **minimum number of characters** from `S` to obtain a **palindrome**  
(a string that reads the same forwards and backwards).

---

### Input

- A single line containing the string `S`.

---

### Output

- The first line should contain the **length of the resulting palindrome**.  
- The second line should contain **one possible palindrome** of that length.  
  If multiple solutions exist, output **any** of them.

---
