### Problem 0.2 — Binary Strings with Exactly K Ones (Simple Version)

**Input file:** standard input  
**Output file:** standard output  
**Time limit:** 1 s  
**Memory limit:** 256 MB

You are given an integer `N`. Your task is to determine how many binary strings of length **N** contain **exactly K ones**.

Because the result may be very large, output it modulo **1000000007**.

---

### Input

The first line contains two non-negative integers `N` and `K`:

- `0 <= K <= N <= 1000`

---

### Output

Print a single integer — the number of binary strings of length `N` that contain exactly `K` ones, modulo **1000000007**.
