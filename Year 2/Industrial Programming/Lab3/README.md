# Lab 3 — Working with Strings (Variant 7)

Develop a **console application in Java** to process text input.  
The text consists of multiple lines entered from the keyboard.  
An **empty line** indicates the end of the text input.

## Variant 7:

- Remove all characters from the text **except letters and spaces**.  
- Between consecutive sequences of letters, ensure that there is **at least one space**.
